# PYRo Wiki documentation site

This repository contains only the VitePress documentation site published to GitHub Pages.

## Development

```powershell
npm install
npm run docs:dev
```

Build the site with:

```powershell
npm run docs:build
```

The Worker API, VS Code extension, D1 migrations, and deployment scripts are maintained separately in the sibling `../platform/` directory. The extension pulls this documentation repository through the Worker, so platform source code is intentionally not part of the Wiki content repository.
