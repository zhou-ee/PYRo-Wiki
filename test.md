# 测试文档

这是一个用于测试 AI 文档审查功能的示例文件。你可以将它添加到你的仓库并提交 Pull Request，观察 AI 是否会自动返回审查建议。

## 简介

本项目的目标是搭建一个基于 GitHub Pages 的技术文档站点，分享嵌入式和前端开发经验。但是我们有时候会写出一些不合规范的文档，需要 AI 来帮忙检查。

### 快速上手

请按照以下步骤操作：

1. 克隆仓库
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 打开浏览器访问 `http://localhost:3000`

## 常见问题

**问：我遇到了一个报错怎么办？**  
答：请检查你的 Node.js 版本，本项目需要 **Node.js 18** 或更高版本。如果你的版本过低，请升级。
